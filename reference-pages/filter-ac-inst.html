FILTER-AC-INST

[GENERIC-FUNCTION]

Function arguments:

  - NOTELIST List of notes (THIS OBJECT DON'T READ LISTS OF LIST.) [default = (6000 6530 7203 5049)]
  - APPROX Cents aproximation of a tempered note. [default = 10]
  - TEMPERAMENT Tempered scale used to compare the note list. [default = 2]

Description:

Filter of notes that can be played by an acoustic instrument with quarte tones, eight-tones, or others.
