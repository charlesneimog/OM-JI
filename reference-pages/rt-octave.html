RT-OCTAVE

[GENERIC-FUNCTION]

Function arguments:

  - FRAQ list of ratios [default = 3/2]
&optional  
  - OCTAVE 2 for one octave; 4 for 2 octaves; 8 for 3; etc... [default = 2]

Description:

This object reduce ratios for one octave (or two or others; optional function), your function is similar to the object octave-reduce, but it operates in ratios domain.
