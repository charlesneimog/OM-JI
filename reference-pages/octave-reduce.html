OCTAVE-REDUCE

[GENERIC-FUNCTION]
OM-JI
Function Reference
ARGUMENTS:
  - note List of midicents. [default = (4800 7200 6000)]
  - grave Bass-note: This note will be the bass-note, so if you use 4800 5000, and use the Bass-note 6500, the result will be (6500 6700) not (6000 6200). For this result use, the object range-reduce. [default = 6000]

Description:

This object reduces a list of notes in a determinate space of octaves. The note of the first inlet will be the more down note allowed, and in the second inlet must be the octaves of the higher notes that will be allowed.
