OCTAVE-REDUCE

[GENERIC-FUNCTION]

Function arguments:

  - NOTE List of midicents. [default = (4800 7200 6000)]
  - OCTAVE Octaves of this reduction. [default = 1]

Description:

This object reduces a list of notes in a determinate space of octaves. The note of the first inlet will be the more down note allowed, and in the second inlet must be the octaves of the higher notes that will be allowed.
