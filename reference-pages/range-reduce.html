RANGE-REDUCE

[GENERIC-FUNCTION]
OM-JI
Function Reference
ARGUMENTS:
  - notelist List of midicents [default = (4800 7200 6000)]
  - grave The lowest note. [default = 6000]
  - aguda The highest note. [default = 7902]
&optional
  
  - octave-r It must be used in the reduction of the pitch-range. In which, the reduction will be by + or - of 1200 cents (1), 2400 cents (2), 3600 cents (3); etc... [default = 1]

Description:

This object reduce a list of notes in a determinate space. The note of the first inlet will be the more lowest note allowed, and the note of the second inlet will be the more higher note allowed.
